# go-itquiz-cli
CLI tool for [Easy ITQuiz](https://go-itquiz.herokuapp.com/)

# Usage
```
$ ./go-itquiz-cli
```

# Example
```bash
$ ./go-itquiz-cli
# Question.1
John Mauchly と J. Presper Eckert によって ENIAC の後継として開発されたプログラム内蔵方式を採用したコンピュータは何か。
 1. Zuse Z3
 2. Harvard Mark I
 3. ABC
 4. EDVAC
Input Number of Answer: 4
Correct!!
...
```
